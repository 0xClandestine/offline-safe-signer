# ⛽ Air-gapped Safe Signer

Should go without saying, but this code is intended to be used offline on an air-gapped device exclusively.

```
python3 sign_safe_tx.py

Enter safe tx hash: ...
Enter keystore path: ...
Enter keystore password: ...
```
Console is cleared to prevent your camera from seeing any sensitive data.
